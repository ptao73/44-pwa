# 🛩️ 44-pwa — 1944 经典空战的 iPhone 原生 3D 版

> 纯前端、PWA、WebGPU 加速、离线可玩 —— 专为 Safari 17+/iOS 17+ 优化。

## ✨ 特性
- ✅ **真 3D 飞行**：Three.js + WebGL2 渲染，P-38 战斗机程序化建模  
- ✅ **iPhone 原生体验**：  
  - 🖐️ 单指拖拽控制飞行方向  
  - ⏳ 长按触发「Loop 全屏清弹」  
  - 📱 双指缩放视角（可选）  
- ✅ **PWA 支持**：添加到主屏幕、离线游玩、推送通知预留
